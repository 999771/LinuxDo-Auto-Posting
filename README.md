# Linux.do 自动浏览助手使用文档

## 脚本简介

Linux.do 自动浏览助手是一个 Tampermonkey 用户脚本，专门用于自动化浏览 Linux.do 论坛帖子。它能够自动识别新帖子、按顺序浏览、避免重复访问，并提供丰富的自定义选项。

## 主要功能

### 🔄 自动浏览
- 自动检测并浏览 `/new` 和 `/unread` 页面的帖子
- 智能去重，避免重复浏览相同帖子
- 可配置的停留时间范围（秒）
- 支持同页面和新标签页两种浏览模式

### ⏸️ 智能控制
- F2 快捷键：暂停/恢复自动浏览
- F3 快捷键：跳过当前帖子
- F4 快捷键：打开设置面板
- 实时状态显示

### 🚫 帖子过滤
- **URL 过滤**：支持完整URL或部分URL匹配
- **关键词过滤**：根据帖子标题关键词跳过
- **外部链接检测**：自动跳过包含指定域名的帖子
- **重定向检测**：自动识别并跳过重定向页面

### ❤️ 互动功能
- 随机点赞功能（可配置概率）
- 智能识别已点赞帖子

### 📊 数据管理
- 完整的浏览历史记录
- 实时日志记录
- 统计信息显示

## 安装方法

### 安装前提条件
1. 安装 Tampermonkey 浏览器扩展
   - [Chrome 版本](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
   - [Firefox 版本](https://addons.mozilla.org/firefox/addon/tampermonkey/)
   - [Edge 版本](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd)

### 安装步骤
1. 复制完整的脚本代码
2. 点击 Tampermonkey 图标，选择"添加新脚本"
3. 删除默认内容，粘贴复制的代码
4. 按 Ctrl+S 保存脚本
5. 访问 https://linux.do/ 即可开始使用

## 使用方法

### 基础使用
1. 访问 `https://linux.do` 或任何支持的页面
2. 脚本会自动开始工作
3. 查看右下角的状态栏了解运行状态

### 设置说明

#### 时间设置
- **最小停留时间**：帖子浏览的最短时间（秒）
- **最大停留时间**：帖子浏览的最长时间（秒）

#### 浏览设置
- **启用自动浏览**：主开关
- **在新标签页打开帖子**：选择浏览模式
- **自动跳转到/new页面**：自动重定向到新帖子页面
- **当/new无帖子时检查/unread**：智能切换页面
- **跳转到/new最少新帖子数**：触发跳转的阈值

#### 点赞设置
- **启用随机点赞功能**：是否自动点赞
- **点赞概率**：0-1之间的概率值

#### 跳过帖子设置
- **自动跳过重定向页面**：检测并跳过重定向
- **跳过包含外部链接的帖子**：根据域名过滤
- **外部域名列表**：需要跳过的域名（每行一个）
- **跳过的帖子URL**：要跳过的具体帖子（支持多种格式）
- **跳过的标题关键词**：标题包含的关键词（不区分大小写）

## 状态栏说明

右下角状态栏显示：
- **运行状态**：⏸️ 已暂停 / ▶️ 运行中
- **已浏览帖子数**：累计浏览的帖子数量
- **浏览模式**：新标签页 / 同页面
- **点赞状态**：✓ 启用 / ✗ 禁用
- **跳过规则数**：当前生效的过滤规则数量
- **最近日志**：最新的操作记录

## 常见问题

### Q: 脚本没有自动运行？
**A:** 检查：
1. Tampermonkey 扩展是否启用
2. 脚本是否在 `https://linux.do` 域名下启用
3. 设置中"启用自动浏览"是否打开

### Q: 如何清除浏览历史？
**A:** 
1. 点击状态栏打开设置
2. 在"操作"部分点击"清除浏览历史"
3. 确认操作

### Q: 如何临时停止脚本？
**A:** 
- 按 F2 快捷键暂停
- 或在设置中关闭"启用自动浏览"

### Q: 跳过规则不生效？
**A:** 检查：
1. URL 格式是否正确
2. 关键词是否拼写正确
3. 外部域名是否完整

### Q: 如何添加当前帖子到跳过列表？
**A:** 
1. 在帖子页面打开设置
2. 点击"将当前帖子添加到跳过列表"
3. 保存设置

## 高级功能

### 精确URL匹配
支持多种URL格式：
- 完整URL：`https://linux.do/t/topic/1164403/1`
- 部分URL：`linux.do/t/topic/1164403`
- 帖子ID：`1164403`

### 外部链接检测
脚本会检查帖子内容中是否包含指定的外部域名链接，自动跳过这些帖子。

### 智能页面切换
当 `/new` 页面帖子数量不足时，自动切换到 `/unread` 页面继续浏览。

## 技术支持
如果遇到问题：
1. 查看脚本日志了解详细错误信息
2. 检查浏览器控制台输出
3. 确保使用最新版本的脚本

## 注意事项

- 请合理配置停留时间，避免对服务器造成压力
- 跳过功能可以帮助过滤不感兴趣的内容
- 定期清理浏览历史可以重新浏览旧帖子
- 使用新标签页模式时，浏览器可能会阻止弹窗，需要允许该域名
